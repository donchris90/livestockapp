# Livestock Farm App (Flask Starter)

## Features
- JWT-based Auth (Flask)
- Marketplace Module
- Agent, Vet, and Logistics Blueprints
- PostgreSQL setup
- Modular route structure

## Setup
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Set environment variables in `.env`
3. Run the server:
   ```bash
   python run.py
   ```
